# 💰 PayWeb | Private Payment Page

A clean and simple webpage to receive payments and support through **Indian payment methods**.

Built using **HTML**, **Tailwind CSS**, and **JavaScript**, this page makes it easy for friends, clients, and supporters to send payments without any backend setup.

---

## 🌟 Features

### For Indian Users ✅
- Displays your **UPI QR code**
- Download QR code option
- Copy UPI ID with one click
- Smart button to open installed UPI apps directly
- Fast and mobile-friendly interface

---

## 🛠️ Built With

- HTML
- Tailwind CSS
- JavaScript (Vanilla)
